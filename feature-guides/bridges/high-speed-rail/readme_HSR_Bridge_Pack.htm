HSR Bridge Pack by choco

This pack contains eight different bridges for the High Speed Rail Project (HSRP):

  • A cable stayed HSR bridge, modelled slightly after the Cooper River Bridge in South Carolina.
  • A simple GHSR concrete pier bridge adapted from pier's used in the Charilaos Trikoupis bridge in Greece.
  • A simple HSR concrete pier bridge adapted from pier's used in the Pulaski Skyway in New Jersey.
  • A basic warren thru-truss GHSR bridge, found along the Shinkansen line from Tokyo to Fukuoka, Japan.
  • A bailey truss HSR bridge, inspired by more various pics around the net.
  • A HSR suspension bridge, slightly modeled after the Kurushima-Kaikyo Bridge in Japan.
  • A cable stayed GHSR bridge, slightly modeled after the Tsurumi Tsubasa Bridge in Japan.
  • A concrete box-girder GHSR bridge used along the InterCityExpress lines in Germany.

You can delete one or more of those bridges if you don't like them, and the fence props included in NetworkAddonMod_HSR_Bridge_Props.dat are optional. However, you should keep this file for the cable stayed HSR bridge, as it also contains the wires.

 Compatibility
This download is only compatible with SimCity 4 "Rush Hour" or SimCity 4 Deluxe.
 Installation

Auto Installation: Double-click the installer file to start the installation. The program will guide you through the process. You can change the default installation path, but we strongly suggest to keep the standard folder in order to make future updates and additions easier.

MAC Users: File Juicer should enable you to use this installer package.

Un-Installing: To un-install the files, simply bulldoze every bridge from this download in your cities and delete the files from the plugins folder afterwards.
 Dependencies

To ensure the proper functionality, make sure the files mentioned below are installed in the plugins folder.

  • Network Addon Mod Essentials from April 2009 or newer
  • Network Addon Mod from January 2009 or newer
  • High Speed Rail Project from April 2008 or newer
 Notes about Nightlighting Effects
In order to see the nightlighting, you must install the EP1-Update 1 for Rush Hour/Sim City 4 Deluxe, and then afterwards, install the SC4-Update for nightlighting (note: you have to be logged in in order to see the download link). If this update isn't installed, you won't see any lighting effects on custom BAT creations.
