Wooden Trestle Rail Bridge by Threestooges

This classic design was derived from the numerous bridges that span smaller gaps and valleys in the early days of US westward expansion. Commonly seen in western movies, this bridge is ideally designed for dry crossings (use the Rain Tool trick) as the base of the bridge may block debris from passing underneath. Nothing will prevent it from being used over water though. This is a rail bridge for the Single Track Rail (STR) by the RAM Team.

Included are also two files that are optional. They will modify the wake effect around the bridge pillars. By default, the bridge doesn't have any wake effect enabled, but you can either use the TS_Wooden_Trestle_STR_All_Wake.dat for having wake at all pillars, or - if you think that this is too much - use the TS_Wooden_Trestle_STR_Alternating_Wake.dat for less wake effect. If you don't like the wake effect at all, simply remove those two files from the plugins folder.

 Compatibility
This download is only compatible with SimCity 4 "Rush Hour" or SimCity 4 Deluxe.
 Installation

Auto Installation: Double-click the installer file to start the installation. The program will guide you through the process. You can change the default installation path, but we strongly suggest to keep the standard folder in order to make future updates and additions easier.

MAC Users: File Juicer should enable you to use this installer package.

Un-Installing: To un-install the files, simply bulldoze every bridge from this download in your cities and delete the files from the plugins folder afterwards.
 Dependencies

To ensure the proper functionality, make sure the files mentioned below are installed in the plugins folder.

  • Network Addon Mod Essentials from March 2009
  • Single Track Rail Mod (STR) by the RAM Team
 Notes about Nightlighting Effects
In order to see the nightlighting, you must install the EP1-Update 1 for Rush Hour/Sim City 4 Deluxe, and then afterwards, install the SC4-Update for nightlighting (note: you have to be logged in in order to see the download link). If this update isn't installed, you won't see any lighting effects on custom BAT creations.
