 Additional Bridges

One-Way Road Bridges

The NAM adds bridge functionality to the One-Way Road network. The following base bridges are included:

  • Level bridge
  • Raised bridge
  • Undertruss bridge
  • Wooden covered bridge
  • Medium suspension bridge
  • Stone arch bridge
  • Small steel arch bridge
  • Cantilever truss bridge
  • Overdeck truss bridge
  • Raised OneWayRoad bridge

OneWayRoad Bridges that have already been built should not have their direction changed. Changing the Flow of Direction for an existing OneWayRoad Bridge, without demolishing it, may result with it not functioning properly. For this reason if you want to change the Flow of Direction for a OneWayRoad bridge, it is best if you demolish the OneWayRoad bridge first & then Re-Build the OneWayRoad Bridge in the direction you want.

Note: The one-way bridges come with one-way textures now.

For more information on the NAM's Additional Bridges for other networks, see the appropriate files in the parent directory of this file. Information on bridges for the RealHighways, the High Speed Rail Project, and the Network Widening Mod can be found in the linked feature guides. Note that this section is still undergoing revision as of the NAM 36 release of September 2017.
