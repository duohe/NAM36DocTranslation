Q&A

The contents of this file aims to answer some of the more common questions and/or problems some users might come across with this mod. It is not intended to replace the Installation Instructions but rather accompany it.

If at any time you have any technical problems &/or questions regarding the contents &/or features of the mod, it is strongly advised that you look through here before deciding to post on the forum/website.


Q: I am new to all this. Where can I find more information about this mod?

A: Anything that you may want to know or may need to know about this mod should be contained in the files that came with this mod. The Read Me contains links to most other information regarding this mod. Some of the files have links within them though, to other related items. It is up to you to make sure you read through the files to find what you may need. That is the whole reason why the documentation was made; to make it easier for people to find the information they want or need first without having to look through forums/websites.

Q: I play SimCity 4 in a different language. Are there any localized versions?

A: There is currently a French, German, and a Spanish translation of the NAM. These are individual packages hosted on different SimCity fansites. Look at ToutSimCity for the French translation, SimCityPlaza for the German translation and on CapitalSimCity for the Spanish translation. If you created another translation, please submit it to the NAM Team, so it can be included in the next version.

Q: I see replicated Menu Icons [or Menu Buttons].

A: This means you have a doubled file in either one of the Plugins folders, that should not be there. This is thee ONLY reason why replicated menu icons might appear in a menu. So search through both Plugins folders more thoroughly until you find the problem file(s).

Q: I have a red arrow and cannot plop the new puzzle pieces.

A: This means you have an outdated version of the NAM Controller in your plugins folder. Please make sure that you installed the latest version, which is either available in the full NAM download, or in the NAM Essentials. Use the NAM uninstaller and the BSC Cleanitol list or Cleanup functionality in the new cross-platform installer to remove outdated files.

Q: Some puzzle pieces or other network items seem to block the traffic.

A: Make sure that you have installed the proper NAM version and NAM Controller. There's a Right Hand Drive (RHD) version of the NAM that is suitable for areas where the cars drive on the right side of the road (i. e. the USA or Germany), and a special Left Hand Drive (LHD) version that is intended for the UK or Japanese versions of SimCity 4.

Q: I have just downloaded the newest version, but nothing works or only some things work & some things do not work.

A: 95% of the time, player issues are a result of users not reading the Installation Instructions properly or not at all. If some thing does not work, it is most likely because of a file conflict with the mod &/or the files have not been installed correctly. So you will need to read the Installation Instructions specifically the section that deals with Compatibility & also What files to use; & then look through the Plugins folder to ensure no other files conflict with this mod & that the files in this mod are installed properly.

Q: I have tried everything, but I still can not get anything to work or only some things work.

A: If you have tried everything, then this means you would have tried, &/or need to try, the following; temporarily remove anything and everything from both Plugins folders, except for this mod & the games 5 original BackGround.PNG files; then Start the game & Start a blank New City and then try out the different items that this mod contains. If this is done properly it will ensure that the mod works, at which point progressively place any other files [BATs/Lots/Mods/etc.] back into the Plugins folder, progressively testing the mod [in a Blank city] to make sure it still works. If at any point the mod no longer works, then that means a file conflict has occurred & you will need to pin-point exactly what file(s) is causing the problem.

Q: I have an old Transit_BugFix_ExpandedOptions file, and I want to keep it because of its features.

A: Remove/delete it from the Plugins folder. Anything & everything in that file [among other things] is included as part of the Network Addon Mod. The BSC Cleanitol file that is included in the NAM download will help you with this task.

Q: Can I use one of the included Traffic &/or Automata Plugin files in this mod & nothing else if I wanted to?

A: Yes. The main reason the Traffic & Automata Plugin files have been included is to help towards adding functionality for certain aspects contained as part of the Network Addon Mod (PLEASE NOTE: If you have used the Installer to install the NAM, the NAM core files will be installed by default).

Q: I use this mod & now have problems of some sort with my game.

A: From time to time new additions included in this mod may affect existing items in/with the game. If you experience any pathing, automata &/or any other problems with any of the networks, &/or items contained in this mod, with the game when using this mod; in the first instance recheck your installation of this mod by following the Installation Instructions & make sure you’ve read the History & Technical Notes for any other relevant information. If a problem persists, in the next instance properly check, completely bulldoze & then rebuild the problem network tiles/items in question. The game should then be given some time to update as a result. If a problem still continues (even after 12-24 months game time) then this may indicate an issue with item(s) in question, in which case please report the problem with specific details so it can be looked into.

Q: I experience game crashing or my game crashes to the desktop when using this mod.

A: There is a known issue with the game that can cause this if certain conditions exist &/or are triggered. Please read the Technical Notes for more details.

Q: I can't place my GLR stations on the draggable GLR anymore.

A: The existing GLR stations have to be modified for the usage of draggable GLR. A patch for most stations will be available separately. It's not necessary to bulldoze stations that are already placed in your cities, though.

Q: I use this mod & now some or all of my Transit Enabled Lots no longer work properly.

A: This mod contains a fix that affects Transit Enabled Lots for Left-Hand Version players. Please read the History & Technical Notes for details regarding this fix.

Q: I use this mod, & now some of my buildings have messed-up textures on them, what gives?

A: This is most likely due to a file conflict. The various parts of this mod have been made using a particular & necessary ID system for the network items; and the only time anything should or would conflict with this mod is if there is another file that is conflicting with it. This would only be due to a custom Mod/Lot/Bat file, as most of the files in this mod have been made so that they do not have conflicting issues with the games original files.

Q: Help! I've installed the new NAM version, and now some NAM items are missing in my cities!

A: Most likely, those items are contained in optional plugins. When installing the NAM, make sure to select the same plugins that you installed with the last version of the NAM. The GLR and double-decker puzzle pieces have been made optional now, and the additional bridges that are using custom models are available as separate plugins that are not longer included in the main NAM download. You'll have to download and install those separately.

Q: I can not get U-Drive-It for Elevated Rail to work.

A: Read the Contents file as it explains how & what is necessary in order to access U-Drive-It for Elevated Rail.

Q: When is this mod going to be complete &/or come out of development stage?

A: Maybe when Maxis releases the next version of SimCity? The files associated with this mod make it possible to continue adding certain additional network items for some time still.

Q: What is the ANT item?

A: The ANT [Additional Network Tool] item in the mod is actually an unfinished item that Maxis left in the games files. Technically it is separate from the other networks, with its own capacity and speeds. As of January 2008, the ANT Plugin has been discontinued and fully integrated into the RealHighway Mod (RHW; released separately). Please delete the NetworkAddonMod_ANT_Plugin_Beta.dat from your plugins folder.

Q: I have installed additional bridges, but they don't show up when I try to build them in the game.

A: Make sure to download the latest version of the NAM Essentials. Since the additional bridges are optional plugins, the functionality may not be included in the NAM core files.

Q: Can we change any of the networks to something else; for example; make them wider?

A: Anything that involves making a network wider, that requires using up more tiles width-wise, is currently not possible. However this may change in the future. The RealHighway Mod (RHW; released separately) Network Widening Mod (NWM; released separately) do get around this limitation to an extent by using the existing game networks with network override technology to allow wider networks.

Q: When are more Interchanges, Overpasses, Ramps, Puzzle Pieces, & other items going to be released?

A: Any additional items made will be released when those making them are good and ready to release them. It may take weeks, sometimes a lot longer, to make something from beginning to end; and no amount of question asking &/or bothering is going to change this or speed up the process.

Q: When is the mod going to be updated next?

A: When those working on it, are good and ready to. We won't provide any release dates, and besides, we like to surprise people. ;-)

Q: Can I still use Traffic Cop?

A: Please read the Installation Instructions regarding this, as it is covered in detail there.

Q: I use one of the “…Traffic_Plugin” files, and my Sims no longer use Mass Transit or they use Mass Transit to much, what’s going on?

A: Most of the Traffic Plugin files alter the way the game’s Traffic Pathfinding behaves. They work for the better so that Sims now commute from Point1 to Point2 more efficiently while still keeping their commuting preferences in mind. So if Sims are no longer using Mass Transit or using Mass Transit to much in a particular city, this means the Sims have found a more efficient way to get from Point1 to Point2. This can have a great deal to do with how a city [& region] has been structured &/or laid out. Essentially, people should [try to] alter the way they design their cities, and try build with the different zones being spaced further apart more.

Q: When I use this mod, my computer &/or cities run very slow, why?

A: The general contents of this mod, especially many of the Traffic Plugin files which increase the accuracy of the games Traffic Pathfinding, may have an impact on the overall general performance of the game. What this does is increase the Resource usage of a computer, which subsequently has a roll-on effect & is why some people may experience a slowdown of their computer &/or cities. The more resources a computer has, namely faster CPU, the less of a problem this will be [typically]. Modern computers should be able to handle the better accuracy without any problems.

Q: When I build an item contained in this mod, my game &/or computer momentarily freezes &/or locks-up when I build the item, what is going on?

A: This may potentially indicate a problem with the item being built, missing the necessary file(s) it needs to function properly. If this happens to you when building an item contained in this mod, then please report the problem with specific details so it can be looked into.

Q: Why do I get asked to build overpasses all of the time, & can I stop it?

A: This will only occur if you are using the NetworkAddonMod_AUTOPROMPT_Plugin.dat file. There is no other reason why this would be happening. So if you do not want to be asked to build overpasses all of the time; then simply remove this file from the Plugins folder. This mentioned file is no longer supported though & should not be used. The BSC Cleanitol file that is included in the NAM download will help you with this task.

Q: Is there anything I can do to help?

A: Yes. We are constantly looking for others to help with various items, especially Texturing/Skinning newly created models. Currently there is only a hand few of people working on this mod, & those currently involved may not always be [so] involved, so the extra help would be good [& some of us are willing to teach others with some of the aspects involved]. The more help all-round would mean some things may be made quicker. Some people have even helped without asking, going out of their way to make various items for this mod, and have thus earned the right to have their name placed in the Credits.

Q: SimCity is just a game, all this just for a game, are you nuts?

A: Absolutely. :P
