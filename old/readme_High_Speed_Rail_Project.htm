High Speed Rail Project by the NAM Team
The High Speed Rail Project (HSRP) adds two new “model variations” to the network systems. A models variation is a new set of models for a network in this case for monorail. In early 2007, new technology was found by Memo which allowed us to use a single puzzle piece to create a starter for a new set of textures or models for the same network, there are already a few mods that take advantage of this technology, namely draggable GLR, SAM, and RHW. At the moment, there are two new model variations included in this mod, HSR, elevated, and GHSR, a ground version. The original HSRP replaced monorail, at the time this was the best way of implementation, and was created by 3ddz, founded by Murakamon, on the 24th of February 2006, since then there has only been one release, HSRP Beta 1. This mod uses the original models from that release for HSR, and a lowered version for GHSR, no modifications were made to the models.

There are known stability issues with GHSR/HSR reverting to monorail models. There are two known reasons this can happen:

  1. The intersection you have created does not have a HSRP model created for it yet. Any dragged portion after this model will be reverted back to the monorail models.
  2. The intersection you created may be a situation where two intersections or models are next to each other. Some of these stability issues can be worked around by creatively drawing your HSRP networks using multiple Starter Pieces, etc.

Regarding the Traffic Plugins in the NAM, they will work exactly the same as if it were the Maxis Monorail. Please keep in mind that on the HSRP tracks Monorail automata will show up. There are several HSRP automata skins available, which probably look nice on your Monorail tracks as well.
 How to use the HSRP in the game

For those of you who have ever used Draggable GLR, SAM or RHW, it is essentially the same for HSR and GHSR. There are two new icons in the Rail Menu, for HSR pieces and for GHSR (and some extra), the starter piece for each network is at the beginning of each TAB ring.

1) Choose whether you want GHSR or HSR, we will take HSR, but the principle is exactly the same for GHSR. When you’ve decided, click on the HSR icon, the first one out of the two. And plop the first puzzle piece where you want to start HSR. (Don't worry, the screenshots below were taken with the German language plugin, but of course there's also an English language plugin available.)



2) As you can see after you have plopped it, there is two tiles of HSR, now we go and select the monorail tool and drag from the end of the starter piece. The goldish colour is the preview for HSR, be assured this will be changed eventually.



3) And there you go, you’ve built a section of HSR. Make sure you drag at least 6 tiles of HSR before you bulldoze the starter piece. Now you can extend this using the puzzle pieces or dragging curves, splits and diagonals.
 Compatibility

This download is only compatible with SimCity 4 "Rush Hour" or SimCity 4 Deluxe. Please remove the old HSRP Beta 1 version that has been published by 3ddz. A Cleanitol file has been included that will do the work for you. The HSRP should be compatible with the APTX's Shinkansen Mod (bullet train skin), it should work from a modding perspective but it has not been tested, so use at your own risk.
 Installation

Auto Installation

Double-click the installer file to start the installation. The program will guide you through the process. You can change the default installation path, but we strongly suggest to keep the standard folder in order to make future updates and additions easier.

Mac Users

A separate ZIP version of the High Speed Rail Project files is available as well.

Un-Installing

To un-install the files, simply bulldoze every High Speed Rail Project item from this download in your cities and delete the files from the plugins folder afterwards.

 Custom Content Creation for the HSRP
Stations

If you are interested in creating new HSRP stations or refurbishing suitable Rail/GLR stations, then drop into the HSRP thread and the necessary details can be given. Since the HSRP is based on the Monorail network, a HSRP station has to be modded similar to a Monorail station.

Anything Else

Generally anything else will require the cooperation of the NAM Team, but if you have an idea for something that hasn’t been said before, then drop into the HSRP thread and we’ll take a look at it.
