 1. Introduction
The intention of this file is to get the Network Addon Mod & associated components installed & to get it working for you. While the content of this mod may be quite immense, especially to new users, it is important that you read the whole documentation before using the Network Addon Mod for the first time, especially these Installation Instructions. This will reduce the chances of any problems, conflicts, complications, &/or headaches that you might otherwise experience as a result of not reading the documentation.

Note: If you are able to make use of the NAM Installer, it is highly recommended that you do so. The installer can be used instead of having to manually install the NAM & as such some of these steps can be skipped.

The contents of this file &/or any other file(s) may change with future updates without notice.
 2. Compatibility
This Network Addon Mod is for RH/Deluxe with RH/Deluxe Update1. This mod is not intended for anything else other than this.

Note: For Mac users: a Mac compatible RH/Deluxe Update1 would be preferred, but until such time as a Mac compatible RH/Deluxe Update1 is available, the above item will not be applicable. Please contact Aspyr regarding any Mac compatible version of RH/Deluxe Update1.

In order to use this Network Addon (Mod) and associated features, please read the following very carefully.

If you have any of the following Transit related items, mods, bugfixes, files:

  • a previous version of the NetworkAddonMod.dat file
  • a previous version of the NetworkAddonMod_AutoPrompt_Optional.dat file
  • a previous version of the NetworkAddonMod_ANT_Plugin_Beta.dat file
  • any NetworkAddonMod related file that pre-dates this version (check & compare the Date/Time Stamp of the files involved for this, especially if you’re having problems)
  • any outdated versions of the RHW (Rural/Real Highway Mod)
  • any outdated versions of the SAM (Street Addon Mod)
  • any outdated versions of the HSRP (High Speed Rail Project)
  • any outdated versions of the STR (Single Track Rail)
  • any outdated versions of the NWM (Network Widening Mod)
  • any outdated versions of Custom Bridges by jeronij, Fukuda and smoncrie
  • any outdated versions of custom puzzle pieces
  • the7trumpet's transit related mods, bugfixes
  • Redlotus' transit related mods, bugfixes
  • Redlotus' overpasses
  • Darkmatter's transit related mods
  • hailman's Variable Route Buses Plugin
  • you use anyone else’s transit related mods
  • or have modified any transit related files yourself

Please be sure to remove any & all of these above files from the Plugins folder before attempting to use this Network Addon Mod, otherwise conflicts will arise and some items may not work properly or as intended.

The new cross-platform installer introduced in July 2010 has an option to remove any known conflicting plugins. On the tab marked "setup", you will find an option for "cleanup". Click "Start Scan", then use the appropriate buttons to backup and delete the files it finds. This should ensure that your NAM installation works properly out of the box.

This version of the Network Addon Mod includes most of the above mentioned Transit related items, & contains any & all currently known bugfixes made to date (where possible &/or practical), created by the Simtropolis Modd Squad &/or others to date. For a list of Transit Bugfixes, please see the History.

A note about GamePlay/Computer Performance

While this mod should work with the games minimum system requirements, for gameplay/computer performance reasons it is highly recommended that it be used on systems that exceed the games minimum system requirements. This recommendation is made due to the nature & amount of content that this mod adds. Modern computers should be able to run the game fine, though.
 3. Dependencies

Other than those (optional where applicable) files listed under What files to use section further below, which are included in this mod, this mod does not have any dependencies. Some content in this mod though has resulted in community made items being made for them. These community-made items are currently considered enhancements to this mod &/or the game & as such are currently not included at this time. The locations of these community-made items are currently outside the scope of this mod. It is advised to search SC4 fansite's forums &/or download areas for any community-made items that may have been made based on the Contents of this mod.
 4. Assumptions
This Network Addon Mod assumes that your Installation Files of RH/Deluxe [with Update1] on your system is mostly in their original form still. If you have modified the Textures &/or anything else network related at all, then this may affect the look &/or functionality of this mod &/or vica-versa.

This mod assumes that your game has the Official Maxis RH/Deluxe Update1 applied to your game. Please do not ask for a conversion for non-RH/Deluxe/Patch1 version(s), as it will not be made.

Note: For Mac users: a Mac compatible RH/Deluxe Update1 would be preferred, but until such time as a Mac compatible RH/Deluxe Update1 is made available, the above paragraph will not be applicable. Please contact Aspyr regarding any release of a Mac compatible version of RH/Deluxe Update1.
 5. What Mod file(s) to use
Note: If you are able to make use of the NAM Installer, it is highly recommended that you do so. The installer can be used instead of having to manually install the NAM.

Once you have unzipped the downloaded file, there should be several NetworkAddonModx.dat file (where x is a number from 1 to 5), a few "Controller" files and several subfolders. The details of all these files/folders are as follows:

The files contained in this mod should not have their names changed otherwise some items may not work as intended.

5.1 Short method

The ZIP version of the NAM comes with various files in the root folder and several subfolders with additional files. The files in the root folder are mandatory and must be installed. All files that are contained in Additional Bridges and Plugins are optional and not required for the proper functionality of the mod. From the Automata Controller folder and the Traffic Controller folder, you need to pick/use one *.dat file from every of those folder. If you have a left-hand drive version of the game, make sure to install the file in the Left Hand Version Plugin folder. Finally, pick one locale (language) file from the Locale Files folder. Please read below for detailed information regarding these files.

5.2 Long method

NAM Core Files

  • NetworkAddonMod1.dat
  • NetworkAddonMod2.dat
  • NetworkAddonMod3.dat
  • NetworkAddonMod4.dat
  • NetworkAddonMod5.dat

The above mentioned files are the main mod files that contains most of the contents. You must install these files. The former NetworkAddonMod.dat has been broken into these files to make updates easier. These files will be referenced as NAM Resource files.

  • NetworkAddonMod_Props.dat
  • NetworkAddonMod_UI.dat

The above files contain several props and UIs that are used in the NAM. They have been separated from the core files for easier updates.

  • NetworkAddonMod_OverpassUpdate_Maxis_Components_Plugin.dat
  • SFBT_Rail Textures Mod_Darker Textures.dat
  • zzz_Dual_Rail_Texture_Maxis_Upgrade.dat
  • zzz_Dual_Rail_Texture_NAM_Upgrade.dat
  • zzz_Dual_Rail_Texture_STR_Upgrade.dat

The first of the above files contains modified Maxis overpasses that correct the color shift of the underlying networks. All NAM overpasses have been adjusted in the same way. All other files contain modified Rail textures that eliminate the color shift between Maxis and custom Rail textures for the zoom levels 1 to 4.

  • NetworkAddonMod_Bridges_Plugin_Controller.dat
  • NetworkAddonMod_Controller_RIGHT_HAND_DRIVE.dat or NetworkAddonMod_Controller_LEFT_HAND_DRIVE.dat
  • NetworkAddonMod_IndividualNetworkRULs.dat

Without the files listed above, the NAM won't be functional. They have been separated from the contents files in order to make updates easier. If you download any new item that is intended for the NAM, such as additional bridges, the RHW (Rural Highway mod), the SAM (Street Addon Mod), the RAM (Rail Addon mod)/STR (Single Track Rail) or other NAM plugins, you must update these files. They will be available as NAM Essentials download.

  • NetworkAddonMod_Locale_english.dat
  • NetworkAddonMod_Locale_deutsch.dat
  • (NetworkAddonMod_Locale_[language].dat)

The text strings for the NAM items have been separated from the resource files in order to make translations easier. If you want to create other language files, please go ahead and do so. If you submit your file to the NAM Team, it will be included in the next version. Please install the version that is matching your game language. If in doubt, install the English locale file.

While the following two plugin files can be used as is without the above files, if you do have the NAM installed you will need to use one of each of the following:

Read/Click here to see which “…_Traffic_Plugin_...” file you should use.

Read/Click here to see which “…_Automata_Plugin_...” file you should use.

If you have a previous version of a “…_Traffic_Plugin_...” file &/or “…_Automata_Plugin_...” file, it does not have to be removed, you can continue to use it [this will always be the case unless notified otherwise].

Read/Click here to see which "..._Volume_Data_View_..." file you should use.

  • NetworkAddonMod_LEFT_HAND_VERSIONS_ONLY_Required_Additional_Plugin.dat

This file is necessary only for active Left-Hand Versions only, such as UK versions for example. Only use this file if your game is a Left-Hand Version, or Left-Hand Drive Version, & it is actively in Left-Hand Drive mode (i. e. Cars drive on the Left-Hand side of the Road). If you play the game in Left-Hand Drive Mode, this file should be installed as it forms part of & provides additional functional & enhancements for various items contained as part of the NetworkAddon Mod. For technical reasons this file exists as an extra *.dat plugin file. If your game is not Left Hand Version, do not use this additional plugin otherwise problems may arise & some items may not work as intended. Please make sure that this file is located in a subfolder that is loaded last. This can be achieved by adding a "z" to the beginning of the folder name, such as z_Left Hand Plugin.

Optional Plugin Files

While the following files form part of the NAM, these files have been separate from the NAM core files, so they can be selected individually. Note: If you have used some of the features that are contained in these files, please make sure to install them again. All plugins should be placed in a Plugins subfolder of the NAM installation folder.

  • NetworkAddonMod_Additional_Bridges_Plugin.dat

This file allows users to build new styles of bridges. You will find them in the dialog box opened when dragging a network over water. All bridges make use of in-game models and don't require external dependencies. Various other bridges can be downloaded separately.

  • NetworkAddonMod_Custom_Highway_Menu_Plugin_Optional.dat
  • NetworkAddonMod_Custom_Misc_Transport_Menu_Plugin_Optional.dat
  • NetworkAddonMod_Custom_Rail_Menu_Plugin_Optional.dat

The file NetworkAddonMod_Custom_Highway_Menu_Plugin_Optional.dat removes 4 of the games original Highway Menu buttons, while still maintaining the ability to use anything & everything that was attached with these 4 removed menu buttons by using the remaining highway menu buttons. This file [also] modifies 4 of the other (remaining) games original Highway Menus buttons; & anything & everything is still accessible via using the TAB button on the keyboard with these modified menu buttons. Feel free to try this file out, as it can be safely removed at any time [while the game is not running of course] if you so wish, without causing any functional/technical problems [this will always be the case unless notified otherwise]. The other two files change the order of some transit network related icons in the Rail and Misc. Transportation menues. This was done in order to provide quicker access to the draggable network tools.

  • NetworkAddonMod_Diagonal_Streets_Plugin.dat

This file allows users to create Diagonal Streets, using the normal Street Network Tool or the special Diagonal Street puzzle pieces. In addition, it adds several Street helper puzzle pieces to the Roads Menu.

  • NetworkAddonMod_FLUPs_Plugin.dat
  • NetworkAddonMod_FLUPs_Props.dat

This file allows users to create Flexible Underpasses for roads, similar to the current modular overpass puzzle pieces. It can also be used for underground roads and tunnels. It will add a menu button to the Roads Menu.

  • NetworkAddonMod_GroundLightRail_Plugin.dat

This file allows users to create a Ground Light Rail (GLR; tram) network. It will add three menu buttons to the Misc. Transportation Menu which contain GLR puzzle pieces, draggable GLR and GLR-in-Avenue puzzle pieces.

  • NetworkAddonMod_GroundLightRailxRoad_Plugin.dat
  • NetworkAddonMod_GroundLightRailxStreet_Plugin.dat

This file allows users to create a GLR-in-Road (or T-RAM, Tram-in-Road) or GLR-in-Street network. It will add three menu buttons to the Misc. Transportation Menu which contain a variety T-RAM puzzle pieces.

  • NetworkAddonMod_High_ElRail_Plugin.dat

This file will allow you to build double-height Elevated Light Rail networks, such as Elevated Light Rail over Elevated Highway and Elevated Light Rail over Monorail. It will add a menu button to the Misc. Transportation Menu.

  • NetworkAddonMod_High_Monorail_Plugin.dat

This file will allow you to build double-height Monorail networks, such as Monorail over Elevated Highway and Monorail over Elevated Light Rail. It will add a menu button to the Rail Menu.

  • NetworkAddonMod_Light_URail_Puzzle_Plugin.dat

This file will allow you to build "double-decker" networks, such as Elevated Light Rail over Streets, Elevated Light Rail over Roads and Underground (Heavy) Rail. It will add menu buttons to the Rail and Misc. Transportation Menus.

  • NetworkAddonMod_OneWayRoad_Arrows_Reduction_Plugin.dat

This file removes all arrows from straight and diagonal OneWayRoads. However, the arrows remain at intersections, so you're still able to tell the direction of a OneWayRoad. In addition, hovering with the Route Query tool above the OneWayRoad will show the arrows as well.

  • NetworkAddonMod_OneWayRoad_Roundabout_Arrows_Reduction_Plugin.dat

This file removes all arrows from roundabouts that have been created with the OneWayRoad tool. Please do not install this file you haven't installed the OneWayRoad roundabout plugin (see below).

  • NetworkAddonMod_Roundabouts_Avenues_Plugin.dat

This file allows users to plop Avenue Roundabouts. It will add a menu button to the Roads Menu.

  • NetworkAddonMod_Roundabouts_OneWayRoads_Plugin.dat
  • NetworkAddonMod_Roundabouts_Roads_Plugin.dat
  • NetworkAddonMod_Roundabouts_Streets_Plugin.dat

These files allow users to create roundabouts using the normal network tools, where applicable.

  • NetworkAddonMod_TurningLanes_Avenues_Plugin.dat
  • NetworkAddonMod_TurningLanes_Roads_Plugin.dat
  • NetworkAddonMod_TurningLanes_Extension_Plugin.dat

These files allow users to create Turning Lanes using the normal network tools, where applicable.

  • NetworkAddonMod_WideRadiusCurves_Avenues_Plugin.dat
  • NetworkAddonMod_WideRadiusCurves_Rails_Plugin.dat
  • NetworkAddonMod_WideRadiusCurves_Roads_Plugin.dat
  • NetworkAddonMod_WideRadiusCurves_Streets_Plugin.dat

These files allow users to create smooth wide radius curves for the Avenue, Railway, Road and Street network. Each menu button contains a number of puzzle pieces. Many of them can be plopped on existing in-game curves in order to make them smoother.

Obsolete Files

  • NetworkAddonMod_ANT_Plugin_Beta.dat
  • NetworkAddonMod_AutoPrompt_Plugin.dat

The above optional files are no longer supported. If you have a previous version of the above files, then make sure you remove them, otherwise conflicts will arise & some items will not work/function.
 6. Where to Install file(s)
Note: If you are able to make use of the NAM Installer, it is highly recommended that you do so. The installer can be used instead of having to manually install the NAM.

For installation purposes be sure to place any relevant Network Addon Mod related files in the exact folder as it is represented in the ZIP file, otherwise some aspects may not work as intended. This will simply require cutting & pasting the necessary file(s) into the same folders.

Any of the files contained in the NetworkAddon Mod that you use should be placed in:

“…Programs\SimCity 4\Plugins” folder

-or-

“…My Documents\SimCity 4\Plugins” folder.

[or in a sub-folder of these].

Just make sure if you place any of the particular files in either location, that any of the files are placed in the exact same folder and you keep using that same folder location with each and any updates of this mod. Also be sure not to change the names of any of the files, otherwise some items may not work as intended. This will reduce the chances of problems and/or conflicts.

Any files not used should not be placed in the Plugins folder, but rather placed elsewhere, otherwise some items may not work as intended.
 7. Traffic Plugin modification programs (Traffic Cop, NAM Tool etc.)
Since the NAM Traffic Plugins (aka. Traffic Simulators) have been overhauled completely, it is not longer advisable to use older programs like "Traffic Cop" or the "NAM Tool", which are not longer updated anymore. A new program called "Traffic Simulator Configuration Tool" has been created to replace those older programs. It reflects the latest findings regarding the pathfinding accuracy and also provides options to modify the traffic and zoning data views.
 8. Where to find
You will find some of the additional Network items in the games Roads menu, Highways menu, Rails menu, & Miscellaneous menu. For some additional items though you will need to activate them yourself using the different Network tools in order for them to appear &/or be activated. An example of this is the OneWayRoad Bridges; you need to select the OneWayRoad Tool from the Menu and drag the tool across a body of water (or land gauge) before you are able to select which Bridge type you want built. Once you select the OneWayRoad Bridge type from the Selection Screen, the game will then build the bridge along with its cost. Please see the Contents for more details regarding the in-game location of the various items.
 9. Point of Contact / Reporting Bugs & Issues
Before reporting a bug or issue with any of the content of this mod, please be sure to read the Technical Notes first regarding known game issues & limitations. For any Transit bugs &/or problems with any of the items &/or features that this Mod has, please see the Modding - Transit Networks forum at Simtropolis or the NAM forum at SC4Devotion and look for the appropriate thread. When reporting a problem, issue or bug, please include as much detail about the problem as possible, including what NAM (Date) version you have, what other NAM related files are installed & a picture of the problem.
 10. Addendum

If there is a particular item and/or feature about this Mod that you do not like, then it is simply asked that you do not use it. The complexity of the games’ files means that a lot of the items in this mod are only possible if it is all done in the one mod, and as such no custom orders to do something specific will necessarily be done. By all means place a request in the Modding - Transit Networks forum at Simtropolis or the NAM forum at SC4Devotion, and it may be included in a future update if it is possible given time constraints and/or other issues.
