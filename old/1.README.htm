SimCity 4 Network Addon Mod by the NAM Team

The Network Addon Mod (NAM) combines all transportation network-related fixes, additions and new creations that have been released so far. It will add countless new features to the existing network tools, such as new overpasses, highway onramps, intersections, ped-malls, a ground light rail network, roundabouts and turning lanes.

 Compatibility
The Network Addon Mod is only compatible with SimCity 4 with Rush Hour Expansion or SimCity 4 Deluxe. Attempting to use this addon with any other version of SimCity 4 will cause the game to crash upon loading. You also need the SimCity 4 Rush Hour/Deluxe Patch (Version 1.1.638) for proper functionality. More detailed information can be found here.
 Installation

Installation

The Windows version of this mod comes with an installer that will guide you through the installation process. If you are a Mac user or if you still want to install the files manually, extract the files that are included in the ZIP to "\My Documents\SimCity 4\Plugins". It is wise to create a new folder for each download to avoid confusion. More detailed installation instructions can be found here.

Uninstallation

For uninstalling the Network Addon Mod, simply delete the files from the plugins folder. Make sure to remove any NAM content from your cities before uninstallation.
